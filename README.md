# lifeofthestudio

### *what you hear, you see.*

---

**`lifeofthestudio`** is a design ecosystem at the intersection of sound and image —
where audio becomes form, frequency becomes color, and rhythm becomes structure.

This is not a portfolio. It is a living system.

<br/>

---

### ▌ WHAT LIVES HERE

```
/sound-visual-systems      → Max/MSP patches & generative experiments
/graphic-works             → Visual outputs from the studio
/audio-projects            → Sound design, composition, sonic research
```

Each repository is a node in the same network.
They reference each other. They respond to each other.
Nothing here exists in isolation.

---

### ▌ APPROACH

> *Sound is not decoration for image.*
> *Image is not illustration of sound.*
> *They are the same event — perceived differently.*

The studio works across:

- **Graphic Design** — systems-based, conceptually driven visual work
- **Audio Production** — composition, sound design, field recording
- **Generative Systems** — Max/MSP, Pure Data, emergent visual structures

Tools are chosen for what they can *think*, not just what they can render.
